HTML

Navescu vam postupke otvaranja .html fajlova na dva operativna sistema koji se najcesce koriste, a to su Windows i MacOS sistemi.

Windows operativni sistem:

Za kreiranje jednog HTML-a preko najjednostavnijeg Windows text programa "Notepad" neophodno je da se prvo napravi jedan Text Documents. To mozete uraditi desnim klikom na povrsinu dekstopa i odabrati opciju "new" zatim klikom na Text document. Nakon toga pokrenuti taj dokument i tako prazan se snimi pomocu opcije "save as" na zeljenu lokaciju pod zeljenim nazivom ali obavezno navesti .html i snimiti kao all files. Nakon toga taj fajl na zeljenoj lokaciji ce promenuti svoju ikonu u jednu ikonu od vasih web browsera. Pokretanjem tog fajla automatski ce se otvoriti web browser sa vasim HTML fajlom, s tim da ce on za sad biti prazan dok ne odradite sledece HTML kodove...

  • index.html
Ovo je primer jednog HTML fajla.

Nakon uradjenog ovog postupka mozete poceti sa nekim od pocetnih HTML kodova. To radite tako sto cete desnim klikom na ikonicu fajla izabrati opciju "Edit in Notepad". Kada to uradite otvoricete taj fajl u Notepadu i mozete krenuti sa pisanjem vaseg HTML-a.

MacOS operativni sistem:

Na MacOS sistemu kreiranje .html fajla je nesto komplikovanije nego na Windows sistemu. Na MacOS-u se koristi aplikacija za kreiranje textova koja se zove TextEdit. Aplikaciju TextEdit možete otvoriti odabirom opcije Finder -> Applications -> TextEdit. Nakon otvaranja aplikacije potrebno je napraviti određene izmene u podešavanjima kako bi ona na odgovarajući način obavila čuvanje fajla kao HTML dokumenta. Prvo je potrebno promeniti format dokumenta na Plain Text. To se obavlja iz prozora sa podešavanjima, koji se dobija pokretanjem opcije TextEdit -> Preferences

Unutar Preferences prozora i njegovog prvog taba New Document potrebno je čekirati opciju Plain Text

Unutar drugog taba (Open and Save) Preferences prozora potrebno je čekirati opciju Display HTML files as HTML code instead of formatted tex

Nakon opisanih izmena u podešavanjima TextEdit aplikacije, fajl je moguće sačuvati kao HTML dokument pokretanjem opcije File -> Save i izmenom naziva i ekstenzije u prozoru za čuvanje

Na kraju, preostaje samo da se potvrdi promena ekstenzije klikom na dugme Use .html

Nakon uradjenih ovih koraka dobicete kreiran .html fajl. Duplim klikom na kreirani dokument on se automatski otvara unutar podrazumevanog browsera (uglavnom je to Safari), a kako bi se on otvorio unutar TextEdit aplikacije, potrebno je odabrati opciju Open With -> TextEdit, koja se nalazi u kontekstnom meniju koji se dobija desnim klikom na vas html fajl.Nakon toga ce vam se otvoriti vas fajl koji ce u ovom slucaju biti prazan i spreman za pisanje vasih HTML kodova.

Sada cu vam navesti neke od osnovnih HTML kodova:

U ovom slucaju koristicu Windows operativni sistem i njegov text program "Notepad"

  • -"< h1> Text < /h1>" ovaj primer oznacava naslov texta gde < h1 > treba da oznacava pocetak a < /h1 > kraj koda
  • -"< p> zeljeni text < /p>" ovaj primer oznacava paragraf u textu takodje, < p> se stavlja na pocetak, a < /p> na kraj koda
  • -"< ul> se pise kad zelite da pokrenete listu, s tim da ispod treba da se pise kod < li> text < /li> da bi izvrsili nabrajanje stavki na listi. Ako lista ima vise stavki treba da se pise taj kod ispred i na kraju svake stavke. I kad zavrsite nabrajanje stavki neophodno je zatvoriti kod sa < /ul>
Ako zelite da stilizujete text na primer da neke od stavki u listi obojite drugom bojom to mozete uraditi sa sledecim kodom:
  • -"< li style="color: red;" > zeljeni text" sa ovim kodom vas text ce biti obojen u crvenu boju kao na primeru ispod
  • primer bojanja texta u crveno
  • boju texta mozete da birate koju god zelite samo treba nakon "color:" da upisete zeljenu boju
Ova vrsta koda spada u CSS jezik koji sluzi za stilizovanje textova.

Kada uradite sve napisane kodove sa zeljenim tekstom snimite fajl sa klikom u Notepadu na opciju "File" zatim "save" i vas racunar ce automatski sacuvati ceo text u vasem HTML fajlu. Nakon toga otvanjem vaseg fajla otvorice vam se vas HTML u vasem web browseru s tim da ce on sad biti ispunjen vasim zeljenim tekstom.
